Projets PHP — Full Stack

Contexte

Ces projets ont été réalisés dans le cadre d’ateliers de professionnalisation et de projets personnels afin d’apprendre le développement Full Stack avec PHP et MySQL.

Projets Réalisés

P1 — Monoshop

Mini boutique en ligne avec système CRUD d’administration et gestion de catalogue.

  • PHP procédural, MySQL, Bootstrap
  • CRUD : produits, catégories, utilisateurs
  • Connexion sécurisée pour les administrateurs

P2 — Blog

Système de blog complet avec espace admin et espace utilisateur.

  • PHP procédural, MySQL, Bootstrap
  • Gestion des articles, utilisateurs, catégories
  • Système d’inscription, connexion, publication

P3 — FoodOrder

Site de commande de nourriture avec interface responsive et gestion des commandes.

  • PHP, MySQL, Bootstrap
  • CRUD complet (plats, catégories, commandes)
  • Application du modèle MVC simplifié

Aperçus

Synthèse des compétences

  • Conception et gestion de base de données
  • Développement d'applications PHP MySQL (CRUD complet)
  • Gestion d'espaces sécurisés
  • Utilisation de Bootstrap pour le responsive
  • Mise en pratique du modèle MVC simplifié
