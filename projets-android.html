Projets Android — Développement Mobile Kotlin

Contexte

Ces projets Android ont été réalisés durant ma seconde année de BTS SIO SLAM afin de découvrir les bases du développement mobile natif sous Android Studio avec le langage Kotlin.

Projets Réalisés

P1 — ChatEquipeFrance

Application de messagerie en temps réel sur le thème de l'équipe de France.

  • Firebase Auth & Firestore
  • Authentification et chat temps réel
  • RecyclerView & Material Design

P2 — WeatherApp

Application de météo connectée à l'API OpenWeather.

  • API REST avec Retrofit
  • Affichage dynamique avec Picasso
  • Design responsive et minimaliste

P3 — AppBooks

Bibliothèque de livres avec affichage dynamique.

  • RecyclerView personnalisé
  • Adapter custom
  • CardView stylisé

Aperçus

Synthèse

  • Développement mobile en Kotlin
  • Connexion API REST et Firebase
  • Gestion d’interface responsive (Material Design)
  • Maîtrise de RecyclerView, Adapter, et services REST
  • Autonomie dans la réalisation d’applications Android
